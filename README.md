# E-Commerce Frontend

A modern, responsive e-commerce frontend built with React and Vite, consuming REST APIs from the Django backend.

## 🛍️ Features

### Current Features
- **Product Listing** - Browse all products with grid layout
- **Product Cards** - Display product information with images, prices, and ratings
- **Product Details** - View detailed product information with add-to-cart functionality
- **Responsive Design** - Optimized for all screen sizes (mobile, tablet, desktop)
- **Fast Performance** - Built with Vite for lightning-fast development and production builds


## 🚀 Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 19** | UI Library |
| **Vite** | Build Tool & Dev Server |
| **Tailwind CSS** | Styling Framework |
| **React Router DOM** | Client-side Routing |
